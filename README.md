# Maggot Every Hit

Download it via the [latest github release](https://github.com/danatron1/MaggotEveryHitMod-Silksong/releases) or through [thunderstore](https://thunderstore.io/c/hollow-knight-silksong/p/danatron1/Maggot_Every_Hit/)

Mod can be configured through r2modman, enabling or disabling:
* Maggot every hit
* All water is maggot water
